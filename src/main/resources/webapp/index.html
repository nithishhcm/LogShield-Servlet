A full-stack Java web application demonstrating Log4j2 best practices. Sign up, log in — every event is captured in the right log level and routed to the right file.

  • INFO — Successful registration & login recorded in auth.log
  • WARN — Duplicate username or wrong password flagged in auth.log
  • ERROR — Database exception logged with full stack trace in errors.log
  • FATAL — DB unavailable at startup → application exits safely
